Background

Because of the recent fallback of the COVID-19 pandemic, most schools, colleges and universities have moved towards digital and remote learning. Most institutions use the help of products like Zoom to facilitate communication and replicate the classroom environment. Many studies, including the user study that we performed, have shown that applications like Zoom are not preffered by many students and do not effectively replicate the traditional classroom environment. Our goal is to create a system that encourages students and instuctors to use all the features that these applications contain to achieve the best digital learning experience that is the most similar to their regular classroom.

Development Process

We started our development by brainstorming new features that we could add to a Zoom type platform. We then reached the consensus of adding a few features like chat reactions, games, whiteboard as well as a reward system. To represent and complete a prototype we made a p5.js mockup which replicated actions like reacting to a message and playing a game.
